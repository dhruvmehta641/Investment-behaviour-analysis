# Investment Behaviour Analysis Dashboard

## 📊 Project Overview
This project analyzes investor behaviour using an interactive Power BI dashboard.

## 🔍 Key Insights
- Average investor age: 27.8 years
- Total investors analyzed: 40
- Most preferred investment: Gold
- Major investment objective: Capital Appreciation
- Top information source: Financial Consultants
